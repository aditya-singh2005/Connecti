# Connecti — AI/ML Module

AI-powered features for the Connecti privacy-first social reconnection platform.

## Features

### 1. Relationship Strength Engine
Calculates how strong the relationship is between two users (score 0–100) based on:
- Shared school, college, workplace
- Mutual friends count
- Common interests & skills (Jaccard similarity)
- Previous meetups & recency of last meeting
- Semantic profile similarity via sentence-transformer embeddings

### 2. Intelligent Hint Generator
Generates progressive hints (5 levels, vague → specific) before identity reveal.
Covers 6 categories: education, college, workplace, location, interests, skills, and social circles.
**User names are never revealed in hints.**

### 3. Reunion Probability Engine
Predicts how meaningful a reconnection could be (0–100%) using:
- Relationship score, mutual friends, past meetups
- Profile similarity (embeddings), recency
- Sigmoid-normalized weighted model with tiered recommendations

## Quick Start

### 1. Create a virtual environment
```bash
cd connecti-app/ml
python -m venv venv
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the demo
```bash
python demo.py
```

The demo runs all 3 features on 3 user pairs (high/medium/low match) and saves results to `outputs/sample_outputs.json`.

## Data Format

Each user in `data/users.json`:

| Field | Type | Description |
|---|---|---|
| `user_id` | string | Unique identifier |
| `name` | string | Full name |
| `gender` | string | Gender |
| `school` | string | School attended |
| `college` | string | College attended |
| `workplace` | string | Current workplace |
| `city` | string | Current city |
| `interests` | list | List of interests/hobbies |
| `skills` | list | List of technical skills |
| `mutual_friends` | list | User IDs of mutual friends |
| `previous_meetups` | int | Number of past meetups |
| `last_meeting_date` | string/null | ISO date of last meeting |

## Using as an API

```python
from engines.relationship_strength import get_relationship_score
from engines.hint_generator import get_hints
from engines.reunion_probability import get_reunion_probability

# Relationship Score
result = get_relationship_score("u001", "u003")
print(result["relationship_score"])  # 0–100

# Hints
hints = get_hints("u001", "u003", max_level=3)
for h in hints["hints"]:
    print(h["hint"])

# Reunion Probability
reunion = get_reunion_probability("u001", "u003")
print(f"{reunion['reunion_probability']}%")
print(reunion["recommendation"])
```

## Notes

- The sentence-transformer model (`all-MiniLM-L6-v2`, ~80 MB) is downloaded automatically on first run.
- All data is loaded from local JSON — no database or API calls required.
- To replace with live data, modify `utils/data_loader.py` to fetch from your API/database.
